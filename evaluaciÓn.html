EVALUACIÓN

Evaluación del Proyecto.

La evaluación del proyecto se llevará a cabo de forma continua, teniendo en cuenta el proceso y el producto final.

Se evaluarán aspectos como:

  • Comprensión lectora
  • Expresión oral
  • Trabajo cooperativo
  • Uso de herramientas digitales.

Cada actividad del proyecto está vinculada a criterios de evaluación, como:

  • Comprender textos y extraer ideas principales
  • Expresarse oralmente con claridad
  • Producir contenidos digitales
  • Buscar y seleccionar información

Descargar documento de evaluación (Tarea 6.1)

Instrumentos de evaluación

Para evaluar el aprendizaje del alumnado se utilizarán diferentes instrumentos:

  • Rúbricas
  • Observación directa
  • Autoevaluación

Estos instrumentos permitirán valorar tanto el proceso como el resultado final del proyecto.
