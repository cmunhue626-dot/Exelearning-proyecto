GUÍA DIDÁCTICA

Guía Didáctica.

Guía didáctica del proyecto.

Este proyecto está dirigido al alumnado de 4º de Educación Primaria en el área de Lengua Castellana y Literatura.

Tiene como objetivo desarrollar la comprensión lectora, la expresión oral y el trabajo cooperativo mediante el uso de herramientas digitales.

Relación entre tareas y criterios de evaluación.

Las diferentes actividades del proyecto están vinculadas a criterios de evaluación que permiten valorar el aprendizaje del alumnado de forma global.

Se trabajan aspectos como:

  • Comprensión de textos.
  • Expresión oral.
  • Producción de contenidos digitales.
  • Búsqueda de información.

Metodología

Se emplea una metodología activa basada en:

  • Aprendizaje cooperativo.
  • Uso de tecnologías digitales.
  • Participación activa del alumnado.

Atención a la diversidad (DUA)

Se tendrán en cuenta los diferentes ritmos de aprendizaje mediante:

  • Actividades adaptadas.
  • Apoyo visual.
  • Trabajo en grupo.
  • Uso de herramientas digitales accesibles.

Recomendaciones para el profesorado

  1. Planificar previamente las sesiones.
  2. Comprobar los recursos tecnológicos.
  3. Adaptar las actividades según el grupo.
  4. Fomentar la participación de todo el alumnado.
